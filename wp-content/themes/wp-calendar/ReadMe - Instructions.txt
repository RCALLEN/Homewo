Installation Instructions

   1. Download and extract wp-calendar-latest.zip.
   2. Upload the wp-content folder to the folder wordpress 
      is installed. (It is okay to merge/overwrite if asked 
      by your FTP program)
   3. Activate the theme and the wp-calendar plugin from your 
      wordpress admin panel.





Usage Notes

    * WP CALENDAR utilizes a special function to operate. With
      a little poking around you will easily be able to customize
      it to display whatever information you want.
    * WP CALENDAR is configured not to operate without having the
      wp-calendar plugin, so make sure it's installed!
